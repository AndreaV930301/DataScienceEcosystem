# Data Science Tools and Ecosystem

Este repositorio contiene un Jupyter Notebook titulado **Data Science Tools and Ecosystem**. El notebook proporciona un resumen de las herramientas y lenguajes clave utilizados en el campo de la ciencia de datos. 

## Contenido del Notebook

- **Lenguajes de Programación Populares en Data Science**: Una lista de lenguajes comúnmente utilizados por científicos de datos, como Python, R y SQL.
- **Librerías Comunes en Data Science**: Una lista de librerías esenciales en Data Science, incluyendo NumPy, pandas y scikit-learn.
- **Herramientas de Desarrollo en Data Science**: Una tabla con herramientas de desarrollo como Jupyter Notebook, RStudio y Apache Zeppelin.
- **Ejemplos de Operaciones Aritméticas**: Ejemplos sencillos de cómo realizar operaciones aritméticas en Python.
- **Conversión de Unidades**: Un ejemplo de conversión de minutos a horas usando Python.

## Objetivo

Este notebook está diseñado como un recurso educativo para aquellos que están comenzando en el campo de la ciencia de datos y desean entender las herramientas, lenguajes y operaciones básicas que se utilizan en la industria.
